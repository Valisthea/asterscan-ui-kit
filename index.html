Pro tips for the designer

→ Open any HTML in Chrome. Cmd/Ctrl+P to save as PDF, or use a clean-screenshot tool (Cleanshot, Snagit, Polish) to grab the visible frame at native pixel size.
→ Drag the SVG logos directly onto a Figma / Illustrator canvas — fully editable, infinite scale.
→ For Adobe After Effects: open the HTML in a browser, use a screen-recorder plugin to capture the rendered frame, drop into your composition.
→ For Photoshop: take a 1× screenshot at the native size (1200×630, 1500×500, etc.) — the file dimensions match exactly.
→ All HTML files are self-contained — no external CSS / no broken refs — they render identically online and offline.
